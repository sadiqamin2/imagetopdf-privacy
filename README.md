# Privacy Policy Site

This directory contains standalone privacy policy assets intended for the public repository:

image-to-pdf-privacy

These files are designed for hosting with GitHub Pages and for use in Google Play Console policy fields.

## Contents

- index.html
- privacy-policy.md
- README.md

## Publishing Steps

1. Create the public repository:
   image-to-pdf-privacy
2. Copy the contents of privacy-site/ into that repository root.
3. Commit:

   git add .
   git commit -m "docs: add privacy policy website"

4. Push:

   git push origin main

5. Enable GitHub Pages:

   Settings
   -> Pages
   -> Deploy from branch
   -> main
   -> root

## Expected URL

https://sadiqamin2.github.io/image-to-pdf-privacy/

This URL will be used in Google Play Console.

## Note

If you choose a different repository name (for example, imagetopdf-privacy), update the URL accordingly.
